## Installation

### Prerequisites

- [Visual Studio Code](https://code.visualstudio.com/)
- [Neo4j Database](https://neo4j.com/download/)

### Install Extension

#### Option 1: VS Code Marketplace
1. Open VS Code
2. Go to Extensions (`Ctrl+Shift+X`)
3. Search for "Git4Neo"
4. Click Install
